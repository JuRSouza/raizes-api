# Raízes do Nordeste – API Back-end

API REST desenvolvida em Java com Spring Boot para simular o fluxo principal da rede de lanchonetes Raízes do Nordeste.

## Fluxo implementado (MVP)

Autenticação → Criação de pedido → Pagamento mock → Atualização de status

---

## Tecnologias utilizadas

- Java 17
- Spring Boot
- Spring Security
- JWT
- Spring Data JPA
- H2 Database
- Maven

---

## Requisitos

Antes de executar, é necessário ter instalado:

- JDK 17
- Maven 3.8+
- IntelliJ IDEA (opcional)

---

## Como executar o projeto

### 1. Clonar o repositório

```bash
git clone https://github.com/jursouza/raizes-api.git
```
